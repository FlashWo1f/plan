## Tips
1. 要有成长性思维
2. 不要靠动力（motivation）支撑，要靠驱动力（drive，对于想要改变现状的那种强烈渴望）。困难时要不忘初心，想想你的drive。
3. 养成稳定的习惯：记录日常把学习这件事本身也培养成习惯，就跟每天刷牙一样。
4. 不要太在意学习时间，要在意专注程度。
5. 学累了怎么办，不要把自己逼到精疲力尽。
6. 保持健康，多出门，定期锻炼，重视坐姿，科学饮食。
